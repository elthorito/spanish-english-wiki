**Diferencias** 

`Ser y Estar :` (Bueno)

```diff

· SER BUENO : 

─────────────

! Talking about a person : *‘virtuous, upright’*

 e.g. Mi hijo es muy bueno, siempre se porta bien. 



! Talking about a thing : *‘of good quality’, ‘beneficial’*

 e.g. (Good quality) : Mi chaqueta de cuero es buena, me costó muchísimo.

 e.g. (Beneficial) : Hacer deporte es bueno.



· ESTAR BUENO : 

───────────────

! Talking about a person / thing : *‘being healthy or very handsome’*

 e.g. (Being healthy) : Esta sopa está muy buena.

 e.g. (Handsome) : Juan está muy bueno.

```



`Ser y Estar :` (Malo)

```diff

· SER MALO : 

────────────

! Referred to a person : *‘being evil, wicked’*

 e.g. Marta es mala, malísma. No la aguanto.



! Talking about a thing or an action : *‘of bad quality’ or ‘ harmful’*

 e.g. (Bad quality) : Este libro es muy malo. No te lo recomiendo.

 e.g. (Harmful) : Fumar es malo para la salud.



· ESTAR MALO:

─────────────

! Talking about a person : *‘being sick, ill’*

 e.g. Hoy no voy a trabajar porque estoy malo.



! Referred to a food: *‘rotten’*

 e.g. Esta manzana está mala, dame otra.

```
