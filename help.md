**Citero (,,)** Introduction:

`,,<command>`

List of commands:

`,,help` / `,,help <command>`
`,,,<quote>` (`,,fetchquote <quote>`) - Quotes
`,,search <query>` - Search by keyword
`,,edit` - Link to github project
`,,wiki` - Link to Wiki

Example usage:

`,,,tildes3` (show 'tildes3')

`,,search le` (search for le)

`,,search \\*` (List all quotes)
